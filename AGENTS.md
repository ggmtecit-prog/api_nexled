# AGENTS.md

Standing instruction for every AI agent working on this project.

---

Follow project instructions (AGENTS.md/CLAUDE.md), apply Karpathy Guidelines, use RTK and Caveman by default, load every relevant installed skill, then implement, verify, commit, and push.

---

## What this means

**Project instructions** — Read `CLAUDE.md` and this file before touching anything. They override all defaults.

**Karpathy Guidelines** — Write the least code that solves the problem. Delete before you add. Simple > clever.

**RTK** — Prefix all CLI commands with `rtk` (e.g. `rtk git status`, `rtk git diff`, `rtk git push`).

**Caveman** — Dense, blunt code. Single-line functions where possible. No abstractions for one-use code.

**Load every relevant installed skill** — Check available skills and invoke the ones that apply to the task before starting.

**Implement** — Make the change. No half-finished work.

**Verify** — Confirm it works. Check output, check files, check no regressions.

**Commit** — Commit with a clear message. Co-author line required.

**Push** — Push to remote. Don't leave work local.
